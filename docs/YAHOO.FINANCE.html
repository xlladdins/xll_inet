YAHOO.FINANCE function

This article describes the formula syntax of the YAHOO.FINANCE function.

Description

Return url for querying Yahoo! financial data.

Syntax

YAHOO.FINANCE(symbol, start, end, _interval)

symbol is the list of symbols to query
start is the start date.
end is the end date.
_interval is the sampling interval. Default is "1d".

Return URL for Yahoo! finance.
