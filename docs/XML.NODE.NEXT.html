XML.NODE.NEXT function

This article describes the formula syntax of the XML.NODE.NEXT function.

Description

Return the next XML node of a given type.

Syntax

XML.NODE.NEXT(node, _type)

node is a handle to a XML node.
_type is an optional node type to find. Default is any.

If _type is missing the next node is returned. The function XML.NODE.NEXT.ELEMENT(node) is equivalent to XML.NODE.NEXT(node, XML_ELEMENT_NODE()) This is the usual way to walk the document tree.

Return to index.
